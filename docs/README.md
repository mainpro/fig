# 无花果文档

本项目由于构建完整的项目文档，基于`Vitepress`编写，可以编译成静态网页，也可以直接查看。

如果要修改文档内容，可以使用开发服务器预览，执行下面的命令启动开发服务器。

```shell
yarn docs:dev
```

执行下面的命令编译。

```shell
yarn docs:build
```

你也可以直接阅读项目中的文档，包含一下内容：

- [需求文档](./requirement.md)
- [设计文档](./design.md)
- [用户手册](./user-guide.md)
- [开发文档](./development.md)
- [测试用例](./test-case.md)
